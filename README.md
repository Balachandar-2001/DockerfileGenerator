# DockerGen

DockerGen is a simple full-stack web application that generates Dockerfiles for different programming languages.

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- API: REST
- Containerization: Docker (coming next)
- Orchestration: Kubernetes (coming next)

## Features

- Generate Dockerfiles for:
  - Python
  - Node.js
  - Java
